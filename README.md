# 🎮 Hangman Game (Python)

A simple text-based Hangman game built using Python. The player has to guess a hidden word one letter at a time before running out of chances.

## 📌 Features

- Random word selection
- Letter-by-letter guessing
- Maximum of 6 incorrect guesses
- Displays guessed letters
- Win/Lose messages
- Beginner-friendly Python project

## 🛠️ Technologies Used

- Python 3
  
## 🎯 How to Play

- The computer randomly selects a word.
- Guess one letter at a time.
- If the guessed letter is correct, it appears in the word.
- If it's incorrect, you lose one chance.
- You have **6 incorrect guesses**.
- Guess the complete word before your chances run out.
